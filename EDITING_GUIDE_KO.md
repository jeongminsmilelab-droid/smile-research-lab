# SMILE Research Lab 홈페이지 수정 안내

GitHub 웹사이트에서 별도 프로그램 없이 내용을 수정할 수 있습니다.

## 기본 수정 방법

1. GitHub 저장소에서 수정할 파일을 엽니다.
2. 오른쪽 위의 연필 아이콘 **Edit this file**을 누릅니다.
3. 필요한 문구만 수정합니다.
4. 아래의 **Commit changes**를 누릅니다.
5. `main` 브랜치에 저장하면 GitHub Pages가 자동으로 다시 배포됩니다.

## 자주 수정하는 파일

- `content/site-content.ts`
  - 연구실 기본 정보
  - 연구 트랙
  - 논문
  - 수상 목록
  - 팀 구성원
  - 뉴스
- `content/award-details.ts`
  - 수상 실적의 펼쳐보기 상세 설명
- `content/bilingual-content.ts`
  - 논문·수상 제목과 학회명의 영문 번역

## 수정할 때 주의할 점

- 따옴표, 쉼표, 중괄호는 삭제하지 않습니다.
- 새 항목은 같은 목록의 기존 항목 하나를 복사한 뒤 내용만 바꾸는 방식이 안전합니다.
- 영문 제목을 먼저 쓰고, 한국어 원문은 `originalTitle` 또는 번역 파일에 유지합니다.
- 학생 이름의 영문 표기는 본인이 확인한 공식 표기만 사용합니다.
- 저장 후 **Actions** 탭에서 초록색 체크가 표시되면 배포가 완료된 것입니다.

문법 오류로 배포가 실패하더라도 기존 공개 사이트는 유지됩니다. 수정이 어려우면 변경할 문구를 ChatGPT에 전달해 업데이트할 수 있습니다.
